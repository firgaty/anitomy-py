# Anitomy-py

Port of [Anitomy](https://github.com/erengy/anitomy) to Python using [SWIG](http://swig.org/).

## Build

__Requirements__:

- SWIG
- C++ compiler
- Python:
    - setuptools
    - distutils

__To build__:

```sh
python setup.py build
```

## Installation

Using PIP:

```sh
pip install .
```
